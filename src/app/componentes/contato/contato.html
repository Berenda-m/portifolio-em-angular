Sobre este projeto

Este projeto foi desenvolvido para ser usado como um treino para a matéria de Angular no curso de Front-End oferecido pela BluSoft em parceria com o SENAI. O objetivo é treinar a criação e organização de páginas da web usando o angular para dividir todos os elementos com componentes e como usar um padrão de header e footer para todas as páginas da web deste projeto.
